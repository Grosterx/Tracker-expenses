# Трекер расходов 

## Описание
Это учебный проект — простое веб-приложение для отслеживания личных расходов, разработанное для демонстрации основных навыков в HTML, CSS и JavaScript. Проект позволяет добавлять расходы по категориям, просматривать их в виде списка карточек, фильтровать по категориям и видеть общую сумму расходов. Данные сохраняются в локальном хранилище браузера (localStorage), так что они не теряются при перезагрузке страницы.

Приложение разработано на чистом HTML, CSS (с использованием Bootstrap для стилей) и JavaScript. Оно адаптивно и работает на мобильных устройствах. Этот проект идеален для начинающих разработчиков, желающих попрактиковаться в манипуляции DOM, работе с localStorage и интеграции CSS-фреймворков.

## Цели обучения
- Освоить основы HTML5 и семантической разметки.
- Научиться стилизации с помощью Bootstrap и кастомного CSS.
- Практиковать JavaScript для динамического взаимодействия с DOM (добавление, удаление, фильтрация элементов).
- Работать с localStorage для сохранения данных на клиенте.
- Разработать адаптивный дизайн для мобильных устройств.
- Понять принципы разработки простых веб-приложений без серверной части.

## Функции
- **Добавление расходов**: Форма для ввода описания, суммы и категории с валидацией.
- **Просмотр расходов**: Список расходов в виде карточек с возможностью удаления.
- **Фильтрация**: Кнопки для фильтрации по категориям (Все, Еда, Транспорт, Развлечения, Здоровье, Другое).
- **Статистика**: Отображение общей суммы всех расходов.
- **Сохранение данных**: Автоматическое сохранение в localStorage.
- **Адаптивный дизайн**: Использует Bootstrap для корректного отображения на разных устройствах.

## Как запустить
1. Скачайте или клонируйте репозиторий с проектом.
2. Убедитесь, что у вас есть файлы: `index.html` и `script.js`.
3. Откройте файл `index.html` в любом современном веб-браузере (Chrome, Firefox, Safari и т.д.).
4. Приложение запустится локально. Нет необходимости в сервере, так как это статический сайт.

## Структура файлов
- `index.html`: Основной HTML-файл с разметкой и подключением Bootstrap.
- `script.js`: JavaScript-код для логики приложения (добавление, отображение, фильтрация, сохранение).

## Использование
1. **Добавление расхода**:
   - Заполните форму в разделе "Добавить новый расход".
   - Введите описание (например, "Обед в кафе").
   - Укажите сумму в рублях (положительное число).
   - Выберите категорию из списка.
   - Нажмите "Добавить расход".

2. **Просмотр и управление**:
   - Расходы отображаются в виде карточек ниже.
   - Каждая карточка показывает описание, сумму и категорию.
   - Нажмите "Удалить" на карточке, чтобы удалить расход.

3. **Фильтрация**:
   - Используйте кнопки фильтрации для отображения расходов только по выбранной категории или всех.

4. **Статистика**:
   - В разделе "Статистика" отображается общая сумма всех расходов.

## Технологии
- **HTML5**: Структура страницы.
- **CSS3**: Стилизация с использованием Bootstrap 5.3.2.
- **JavaScript (ES6+)**: Логика приложения, работа с DOM и localStorage.
- **Bootstrap**: Для адаптивного дизайна и компонентов (карточки, формы, кнопки).
- **Bootstrap Icons**: Иконки для навигации.

## Совместимость
- Поддерживает все современные браузеры с поддержкой ES6.
- Работает оффлайн, так как не требует интернета (кроме загрузки Bootstrap с CDN при первом запуске).
